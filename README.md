# minikube on Linux arm64

minikube setup based on ubuntu-20.04-arm64 running as a Vagrant Virtual Machine with the _Parallels_ provider.

## Resources
 - Memory: 2.5 GB
 - CPU: 2

## Applications
- Docker
- Docker Compose
- minikube

## Just start

```BASH
minikube start --kubernetes-version=v1.24.7
```
